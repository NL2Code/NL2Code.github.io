- 🔑Supported Programming Language (PL): Ghostwriter performs best with `JavaScript` and `Python` code, but it supports 16 languages in total. - 🔆Supported Feature: We’re developing Ghostwriter to be faster, more powerful, and more accessible than any comparable offering. Our product features are constantly getting better and faster.
  + Complete Code: Ghostwriter's flagship feature. Ghostwriter leverages context to provide code suggestions as you're typing. Press `tab` to accept suggestions.
  + Explain Code: Highlight code, and Ghostwriter will walk through it step-by-step in plain English.
  + Transform Code: Transform Code allows users to highlight blocks of code, describe the adjustments needed (like changing variable names), and Ghostwriter will provide a suggested rewrite of the code block.
  + Generate Code: Provide a natural language prompt, and let Ghostwriter generate code to help solve your problem.
