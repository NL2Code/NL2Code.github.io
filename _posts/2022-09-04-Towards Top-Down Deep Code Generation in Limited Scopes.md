## 🌸Method

- 📙Title: [Towards Top-Down Deep Code Generation in Limited Scopes](https://arxiv.org/pdf/2209.01566.pdf)
- 📚Publisher/Date: `Arxiv/2022`
- 🏠Author Affiliation: University of Zurich
- 🔗URL: `None`
- 💕Contribution
  + To summarize, to overcome a few restrictions of neural methods, we suggest building a code taxonomy in light of code information. And then, we propose a semantic-aware framework, taking the semantic pyramid as its mental model, for top-down code generation.
  + As an exploratory work, top-down code generation is novel but faces challenges. For example, human-machine interactions could be very heavy in some cases where requirements are hard to capture or fulfill. Thus, we focus on realizing the proposed framework in limited scopes and seek continuous evolutions thereafter.
