- 💕Contribution
  + In this work we focus on jointly modeling both source code (Python) and concomitant natural language documentation (docstrings) with transformers, through the study of dual tasks: generating method code bodies from signatures and docstrings, and generating docstrings from signatures and method code bodies.
  + we present a multi-mode translation method resulting in PyMT5, the Python method text-to-text transfer transformer. Our single model can both learn code/language generation and understand the relationships between them.
